# AfterFaire 2027

Profitability-first planning and marketing materials for an intimate Renaissance
Faire after-hours gathering at Merveilles Pleasure Gardens, approximately ten
miles south of the Colorado Renaissance Festival.

## Project files

- `business-plan.md` — operating, marketing, permitting, and financial plan
- `financial-model.csv` — auditable scenario summary
- `index.html` — public campaign concept and interactive break-even calculator
- `styles.css` / `script.js` — site presentation and calculator behavior
- `assets/` — original campaign illustrations

## Preview

Open `index.html` in a browser, or serve the directory locally:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Planning status

All 2027 dates, capacities, camping, food, beverage, and entertainment details
remain provisional until contracts and government approvals are complete. The
business plan identifies the gates that must be cleared before public ticket sales.
